About GenshinCo.de

A fan-made code tracker. Nothing more, nothing less.

I got tired of missing codes because I wasn't watching the right stream at the right time. So I made this page. It's a simple list of active Genshin Impact redeem codes, updated whenever new ones come out. No accounts, no noise.

How it works

Codes get picked up from HoYoverse's livestreams, their social posts, and patch notes. Each one is listed with what it gives so you can decide if it's worth the 30-second detour. The "Redeem" button goes straight to the HoYoverse gift page with the code pre-filled, you just log in and confirm. This site never touches your account.

Not affiliated with HoYoverse

This is a personal fan project, not an official HoYoverse product. Genshin Impact and everything related to it belongs to HoYoverse / miHoYo.

Codes expire, sometimes within hours. I do my best to keep the list up to date, but if one doesn't work, it's probably already gone. The official HoYoverse site is always the source of truth.

How codes are collected

This site runs on automation. A GitHub Action scrapes known sources for new codes every 6 hours -livestreams, official posts, patch notes, and community reports. When new codes are found, they're automatically committed to the repository and the site updates instantly. No manual work required for most codes.

Occasionally, codes pop up from obscure sources (a random tweet, a stream chat message, an email) that the automation doesn't catch. Those get added manually. The pipeline handles the volume; the humans catch the stragglers.

Open source

The entire site is open source on GitHub. No hidden code, no proprietary magic. You can fork it, run it locally, contribute features, or just peek under the hood to see how it works. If you spot a bug, have an idea, or want to help, open an issue or submit a pull request.

Contact

Have a question, found a bug, or want to suggest a feature? Open an issue on GitHub. That's where I track everything and where the community can chime in too.
