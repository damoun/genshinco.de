How to Redeem Genshin Impact Codes

Step-by-step instructions for web and in-game redemption

Genshin Impact codes can be redeemed in two ways: through the official HoYoverse website or directly in-game. Both methods take under a minute. You'll need an active Genshin Impact account at Adventure Rank 10 or above.

Redeem via the HoYoverse Website

  1. Visit the official gift page at genshin.hoyoverse.com/en/gift - or click "Redeem" on any code at GenshinCo.de, which pre-fills the code.
  2. Log in with your HoYoverse account.
  3. Select your server region (America, Europe, Asia, or TW/HK/MO).
  4. Choose the character you want to receive rewards on.
  5. Paste or type the code and click Redeem.

Rewards appear in your in-game mailbox within a few minutes.

Redeem In-Game

  1. Open the game and press Esc (PC) or tap the Paimon menu.
  2. Go to Settings (gear icon).
  3. Select the Account tab.
  4. Click "Redeem Now" and enter the code.

This method doesn't require a browser and works on all platforms.

Common Errors and How to Fix Them

  • "Redemption code has been used" - Each code can only be used once per account.
  • "Redemption code has expired" - Most codes expire within 12-48 hours.
  • "Redemption code is invalid" - Double-check for typos; codes are case-sensitive.
  • "Adventure Rank is not high enough" - You need at least AR 10 to redeem codes.
  • "Server region mismatch" - Make sure you selected the correct server.

Tips for Getting Codes First

  • Bookmark GenshinCo.de - codes are posted here as soon as they're discovered.
  • Watch HoYoverse's version livestreams on YouTube or Twitch - 3 codes are usually revealed.
  • Follow @GenshinImpact on social media for event codes.
  • Check during patch days (every ~6 weeks) when new codes typically drop.

Frequently Asked Questions

How many codes can I redeem?

There's no limit. Redeem every active code you find.

Do codes work across all platforms?

Yes. Codes are tied to your account, not your platform (PC, mobile, PlayStation).

Can I share codes with friends?

Yes. Codes work for everyone until they expire. Share them freely.
